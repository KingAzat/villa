## 📖 Description

**Africa Premium Properties** est une plateforme web moderne et professionnelle dédiée à l'immobilier de luxe en Afrique. Ce site présente des propriétés exclusives dans les plus grandes villes africaines (Abidjan, Dakar, Douala) avec un design élégant et une expérience utilisateur optimisée.

Le projet a été entièrement adapté pour le marché africain avec :
- 🇫🇷 Interface 100% en français
- 💰 Prix affichés en Franc CFA (FCFA)
- 🏠 Localisation africaine authentique
- 🎨 Design premium aux couleurs de l'excellence africaine

## 🚀 Installation et Utilisation

### Prérequis
- Navigateur web moderne (Chrome, Firefox, Safari, Edge)
- Serveur web local (optionnel, peut s'ouvrir directement)

### Démarrage Rapide

1. **Cloner le repository**
```bash
git clone <url-du-repository>
cd africa-premium-properties
```

2. **Ouvrir le projet**
```bash
# Option 1: Ouvrir directement dans le navigateur
open index.html

# Option 2: Utiliser un serveur local (recommandé)
# Avec Python
python -m http.server 8000

# Avec Node.js (npx)
npx serve

# Puis accéder à http://localhost:8000
```

3. **Personnalisation**
- Modifier `assets/css/templatemo-villa-agency.css` pour les styles
- Éditer les fichiers HTML pour le contenu
- Remplacer les images dans `assets/images/`

## 🎨 Personnalisation

### Changer les Couleurs
Dans `assets/css/templatemo-villa-agency.css`, modifiez les variables CSS :
```css
:root {
  --primary-color: #c9a227;  /* Or */
  --secondary-color: #1a1a2e; /* Bleu marine */
}
```

### Ajouter des Propriétés
Copiez la structure d'une propriété existante dans `properties.html` et adaptez :
- Titre et description
- Prix en FCFA
- Localisation (ville, quartier)
- Caractéristiques (chambres, salles de bain, superficie)
- Images
